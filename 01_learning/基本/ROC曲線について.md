## 2. 確定した定義・式（抜粋）

### ROC curve
- フルスペル：Receiver Operating Characteristic curve
- 意味：しきい値を動かしたときの **TPR（縦）** と **FPR（横）** の関係を表す曲線
- 軸：
  - 横軸：FPR（False Positive Rate）
  - 縦軸：TPR（True Positive Rate）

### 二クラス分類
- 英語：binary classification
- 意味：判定が2値（例：陽性/陰性、Yes/No）

### TPR
- フルスペル：True Positive Rate
- 意味：実際に陽性のものを、陽性と正しく判定できた割合
- 式：TPR = TP / (TP + FN)

### FPR
- フルスペル：False Positive Rate
- 意味：実際に陰性のものを、誤って陽性と判定した割合（誤検知率）
- 式：FPR = FP / (FP + TN)

### 混同行列の要素
- TP（True Positive）：陽性を陽性と判定（正解）
- FP（False Positive）：陰性を陽性と判定（誤り）
- TN（True Negative）：陰性を陰性と判定（正解）
- FN（False Negative）：陽性を陰性と判定（誤り）

### AUC
- フルスペル：Area Under the Curve
- 意味：ROC曲線の下面積（大きいほど良い）
- 目安：
  - 0.5：当てずっぽうレベル
  - 1：理想

## 3. 暗記カード（最小セット）

- 覚える式：
  - FPR = FP / (FP + TN)
  - TPR = TP / (TP + FN)

- 覚える対応：
  - ROC：横軸FPR、縦軸TPR

- 覚える判定ルール：
  - 左上に寄るほど良い
  - 対角線は当てずっぽう（AUC ≈ 0.5）
  - AUCは大きいほど良い（最大1）
